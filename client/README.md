#  Digital Life Lessons

**Live Site:** https://your-live-site-url.com

Digital Life Lessons is a full-stack web platform where users can share meaningful life experiences, learn from others, and grow together. It bridges the gap between fleeting thoughts and lasting legacy, allowing users to document personal growth, career insights, and emotional realizations in a structured, community-driven environment. The platform includes user dashboards, admin controls, analytics, featured lessons, and a modern responsive UI.

---

##  Features
* **Premium Insights with Stripe:** A seamless upgrade path using Stripe Checkout (test mode) that unlocks "Premium-only" content and exclusive platform features for a lifetime access fee of ৳1500.
*  Users can create, edit, and share life lessons with categories, tones, and images
*  Save lessons, like posts, and explore featured or trending content
*  Admin dashboard with analytics, reports handling, and contributor insights
*  Dynamic charts showing lesson growth, user growth, and platform activity
*  Premium UI with TailwindCSS, glassmorphism effects, and animations
*  Authentication with Firebase for secure login and role-based access
*  Fast API architecture with optimized MongoDB queries and aggregated dashboard endpoints
